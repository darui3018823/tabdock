    Tabdockについて

    darui3018823
    Student
    Tabdock: UnknownVersion
    Mail: contact@daruks.com
    GitHub Repository: darui3018823/tabdock

    Tabdockは、使われなくなったタブレットを情報表示端末として再活用するためのWebベースのダッシュボードです。

    • 予定管理、天気、システムステータスの可視化
    • クライアント主導型UI＋軽量なローカルAPI接続
    • 自由なウィジェット設計による拡張性

    開発者向けメニュー

    同期・診断

    完全同期

    サーバーから主要データを再取得し、ウィジェットを更新します

    JavaScriptロード状態確認

    主要スクリプトの読込状況と関数の有無をチェックします

    簡易診断

    UI・API・ブラウザ環境をまとめて検証します

    パフォーマンステスト

    DOM生成やネットワーク応答時間を計測します

    メンテナンス

    カレンダー再生成

    祝日・予定・サブスクリプションを再描画します

    ローカルストレージ初期化

    保存済みの設定やキャッシュを削除します

    デバッグログ表示

    最新ログと環境情報を確認・エクスポートします

    画面焼け防止モード

    インタラクションが無い間は暗転させます

    その他ツール

    ログインユーザーのシフト削除

    サーバー上の個人シフトを一括削除します

    ページ再読み込み

    最新の構成に更新します（保存されていない変更は失われます）

    デバッグ情報

    画面焼け防止モード
    画面をタップ、または ESC で終了します
